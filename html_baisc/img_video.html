이미지 태그

이미지 태그는 인라인으로 인라인 요소와 형제관계여야 합니다.

이미지 태그 작성시 필수로 src, alt 속성을 반드시 작성해야합니다.


동영상 태그

동영상태그는 video는 블록태그로 블록과 형제관계여야 합니다.

브라우저 별로 호환되는 확장자를 주의하며 동영상을 연결해야 합니다.

동영상태그의 주요 속성 종류와 뜻

1. autoplay : 자동재생

2. muted : 음소거

3. countrols : 동영상 컨트롤러 보이기

4. loop : 동영상 반복 설정

"안되는 이유는 처음부터 이미지와 비디오를 설정하지 않았기때문이다. 애초에 이미지,비디오가 없으니깐 임시링크를 넣어서도 안되고 코드를 쓰면 안되기 때문!"
