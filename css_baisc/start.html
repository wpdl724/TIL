CSS Style Sheet

CSS 작성 시 외부스타일, 내부스타일 작성방식이 있다. 내부 스타일은 <head> 태그 내에 < style> 태그로 작성하고 외부스타일은 별도의 CSS파일을 생성하여 <link>태그로 연결하는 방법이다

  • 아이디는 CSS에서 #으로 표시한다.
  • 태그는 CSS에서 꺽쇠없이 태그명만 표시한다.
  • 디자인하는 대상이 정해졌을 때
  • 그 대상에 가장 가까운 태그의 태그명 또는 아이디, 클래스를 확인하고 선택자를 작성해야 한다.
자식 em span 안 두번째 자손 em위치

CSS

CSS는 HTML을 꾸미는 디자인언어다.

CSS는 HTML을 먼저 작성하고 CSS를 작성한다.

내부스타일시트

내부 스타일은 head 태그 안에 작성한다.

내부스타일은 다른 HTML에 연결 할 수 없다.

주의사항

속성 값 끝날때마다 세미콜론(;) 작성하기

  • 영문소문자로 클래스, 아이디 작성
  • 공백, 특수문자 불가

CSS 속성색상

CSS에서 작성 할 수 있는 색상 값의 종류

color, background-color

  • 헥사코드
  • #으로 시작하는 6자리 RGB코드 #ffff00
  • 6자리 모두 같은 글자 반복시 3자리로 줄여쓰기 가능
RGBA
rgb와 alpha(A) 투명도를 함게 쓸수 있는 코드
A는 0~1까지의 값만 작성한다. 1은 100%와 같다.
rgba(255,0,0,0.2)란 빨강색 20% 투명도를 뜻한다.

선택자

태그, 아이디, 클래스 선택자

태그와 아이디 또는 클래스가 함께 있을 경우는 아이디, 클래스를 선택자를 먼저 작성한다.

자식, 자손 선택자

특정 부모 요소에 자식, 자손 태그가 중복 되어있을 경우만 >표시를 사용해 자식을 구분한다.

형제 선택자

인접한 형제는 + 기호를 사용한다.

모든 형제는 ~ 기호를 사용한다.
